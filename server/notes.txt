1. Understanding express ? 

const express = require("express"); express -> function
const app = express(); -> here we are calling the function, and it returns an application object.
app.listen(5000);

npm install express
        │
        ▼
Downloads the Express package
        │
        ▼
require("express")
        │
        ▼
Gets the express() function
        │
        ▼
express()
        │
        ▼
Creates the Express Application (app)
        │
        ▼
Configure the application
(Routes • Middleware • Settings)
        │
        ▼
app.listen(PORT)
        │
        ▼
Registers the application with the Operating System on that port
        │
        ▼
Browser sends a request to localhost:PORT
        │
        ▼
Operating System forwards the request to Express
        │
        ▼
Express matches the route and executes the appropriate handler
        │
        ▼
Response is sent back to the browser

3. package.json and package-lock.json ? 
-> package.json = It tells npm WHAT this project needs.
Inside it live all your installed packages (express, dotenv, body-parser etc.)

-> package-lock.json = It records the exact versions of every package that was installed.

package.json
        │
        ▼
Tells npm what to install
        │
        ▼
npm downloads packages
        │
        ▼
node_modules
        │
        ▼
Actual installed libraries

4. Request pipeline 

Request (POST /api/contact)
   │
   ▼
Validation Rules (contactValidation.js) -> Defines fields & formats (.isEmail(), .notEmpty())
   │
   ▼
validate Middleware (validate.js) -> Analyzes verification results
   │
   ├── Invalid (errors exist) → Return 400 Bad Request (Error Response)
   │
   └── Valid (no errors) → next()
                 │
                 ▼
            Controller (contactController.js) -> Runs core logic
                 │
                 ├─► Saves to Database (ContactMessage Model)
                 │
                 └─► Sends Email (Mailer Utility)

5. Nodemailer setup 

Frontend (Form)
      │
      ▼
Express (Backend API)
      │
      ├────────► MongoDB (Message Log)
      │
      └────────► Gmail / SMTP (Email Notification via Nodemailer)

Nodemailer is a Node.js library that allows secure connection to an external email service (SMTP server) using credentials (USER, PASS) to send dynamic HTML or plain-text messages.
